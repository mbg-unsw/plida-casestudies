\begin{frame}{Key lessons}
\begin{itemize}
	\item Bigger data, slower analyses
	\item New tools help: DuckDB
	\item Nightly automatic shutdown: don't forget
	\item Quarterly updates: how to manage?
	\item SYNTHETIC\_AEUID ain't SYNTHETIC\_AEUID
	\item Docs are great but not always complete or up-to-date
\end{itemize}
\end{frame}

\begin{frame}{Reflections/insights for newbies}
\begin{itemize}
	\item DuckDB needs temporary space, request a 1TB X: drive (it's free)
	\item Understand data size and its impact on speed
	\item Generate smaller test data sets for developing and testing your code
	\item Develop reproducible data flow with logs you can show ABS
	\item Census data is cool but has its own quirks (90% linkage, self report, 2021 only, coding)
	\item We all have a lot to learn, share the knowledge!
\end{itemize}
\end{frame}
